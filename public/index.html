Example: Hello Netlify Functions

A hello world example of using Netlify Functions to add serverless functions to your project.

To add Netlify Functions, create a JavaScript, TypeScript, or Go function to your project in a netlify/functions directory. You can customize the address for your functions using Netlify's Redirects API

How to use?

  • Hello World ( /.netlify/functions/hello?name=you )
  • Hello World ( /api/hello?name=you )

  • All Users( GET /api/crud )
  • All Users( GET /api/crud?db=sample_mflix&col=users )
  • Single User Users( GET /api/crud?id={id} )

  • ( POST /api/crud )
    body: { "field1": "value1", "field2": "value2" }

  • ( PUT /api/crud?id={id} )
    body: { "field1": "value1", "field2": "value2" }

  • ( PATCH /api/crud?id={id} )
    body: { "field1": "value1" }

  • ( DELETE /api/crud?id={id} )

  • ( OPTIONS /api/crud ) > CORS

  • ( HEAD /api/crud ) > CORS

All Parameters

  • uri (MongoDB URI)
  • db (MongoDB Database)
  • col (MongoDB Collection)
  • skip (Default: 0)
  • limit (Default: 100)
  • sort (Default: {})
    Usage: sort=name,1*bDay,-1 (1 for 'asc', -1 for 'desc')
  • id
  • q (Search in all fields)
    Usage: q=dude
